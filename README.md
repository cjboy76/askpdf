# AskPDF

A PDF Reader App with integrated chat, read and discuss in real time.

<img alt="hero image" src="public/og.jpeg"  width="100%" height="100%">

### Development Setup

1. Clone the repo
2. Run `npm i`
3. Create a `.env` file and setting up [Google OAuth](https://console.cloud.google.com/apis/credentials), [OpenAI API](https://platform.openai.com/api-keys), please refer to `.env.example`
4. Open http://localhost:3000
